# Pixee Homebrew Tap

Homebrew tap for [Pixee](https://pixee.ai) CLIs.

## Install

### Pixee CLI (current)

The official command-line client for the Pixee platform — the binary distributed by
[`pixee/pixee-cli`](https://github.com/pixee/pixee-cli):

```bash
brew tap pixee/tap
brew install pixee-cli
# or, in one step:
brew install pixee/tap/pixee-cli
```

### Legacy Pixee (Python codemodder CLI)

The retired Python codemodder CLI remains published here for backward compatibility:

```bash
brew install pixee/tap/pixee
```

No new releases are planned for the legacy formula.
